  title: "Docheal"
  description: "Una aplicación web para programar citas con el médico, facilitando una comunicación fácil y eficiente entre pacientes y doctores."
  project_img: "../../../public/assets/turnos.jpg"
  github_repo: "https://github.com/EliasPereyra/docheal"

  summary: "Docheal ofrece una interfaz limpia y una experiencia de usuario fluida, asegurando una programación de citas eficiente tanto para pacientes como para médicos. También proporciona notificaciones móviles para que los pacientes reciban actualizaciones sobre sus turnos. La aplicación web busca hacer del agendamiento de citas médicas un proceso sin complicaciones."
  key_features: 
    - "Registro con correo electrónico y contraseña"
    - "Registro de toda tu información personal y médica"
    - "Solicitar citas con el médico"
    - "Panel con todas tus citas"
    - "Un panel para el administrador donde puede gestionar la información y los turnos de los pacientes"
    - "Diseño responsivo para distintos tamaños de pantalla"
  design_goals: 
    - title: "Interfaz limpia"
      description: "El diseño se basa en buenos principios de UI/UX, lo que lo hace fácil de navegar y comprender."
    - title: "Seguridad y privacidad"
      description: "La aplicación utiliza Zod para validar los datos del usuario, asegurando que solo se envíe información válida al servidor."
    - title: "Enfoque en el front-end"
      description: "Con la implementación de Appwrite (un BaaS), el front-end busca mejorar la experiencia del usuario y hacerla más eficiente."
    - title: "Diseño responsivo"
      description: "Para que los usuarios puedan acceder a la aplicación web desde cualquier dispositivo, sin importar su tamaño."
